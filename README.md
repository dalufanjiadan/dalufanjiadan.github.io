# hexo

### 1. 常用命令

- `hexo generate [-d]`
- `hexo serve [-p port]`
- `hexo deploy [-g]`
- `hexo s -p 5000`

### 在本地运行

- `npm install`
- `hexo s`

### 更新主题

- `git clone --branch v6.0.0 https://github.com/theme-next/hexo-theme-next themes/name`
- 更改`_config.yml` theme

### 新建

- `hexo n post <title>`


### 推送到github

- `hexo d -g`
