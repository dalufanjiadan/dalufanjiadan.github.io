### 查看端口占用

```sh
netstat -ap | grep 8080
kill -9 PID
```

### 运行jar包

```sh
nohup java -jar mba-0.0.1-SNAPSHOT.jar  > nohup.log &
```

### 删除

- 删除文件

    ```sh
    rm -f test
    ```

- 删除文件夹 

    ```sh
    rm -rf test
    ```
