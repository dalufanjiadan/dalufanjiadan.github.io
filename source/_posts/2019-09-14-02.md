### 配置

```json
{
    // 调整窗口的缩放级别
    "window.zoomLevel": 3,
    // 更改terminal
    "terminal.integrated.shell.windows": "D:\\Git\\bin\\bash.exe",
    // 控制树缩进(以像素为单位)
    "workbench.tree.indent": 15,
    // 指定的控制台用于启动Java程序
    "java.debug.settings.console": "internalConsole"

}
```
