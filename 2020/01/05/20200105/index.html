新建并启动

  • docker run

使用 docker run 命令创建容器时，会先检查本地是否存在指定镜像。如果本地不存在该名称的镜像，Docker 就会自动从 Docker Hub 下载镜像并启动一个 Docker 容器。

  • docker attach

很多场景下，使用 docker attach 命令并不方便。当多个窗口同时 attach 到同一个容器时，所有窗口都会同步显示。同理，如果某个窗口发生阻塞，其他窗口也无法执行操作。
